## 🎨 核心功能详解

### 1. PDF基础工具（6个）

**本地处理模式**：
- 100%浏览器端处理，文件不上传
- Web Workers后台处理，UI不阻塞
- 自动内存管理，防止泄漏

**云端处理模式**（Pro/Enterprise）：
- 文件上传→Celery任务→轮询/WebSocket→下载
- 支持大文件和复杂操作
- 实时进度反馈

### 2. OCR文字识别

**支持语言（10种）**：
- 英语、简体中文、繁体中文
- 日语、韩语、法语
- 德语、西班牙语、俄语、阿拉伯语

**技术实现**：
- 前端：OCRPDF.vue（850+行）
- 后端：Tesseract + pdf2image + PIL
- 结果：文本+置信度+页数统计

### 3. Office转PDF

**支持格式**：
- Word: .docx, .doc
- Excel: .xlsx, .xls  
- PowerPoint: .pptx, .ppt

**技术实现**：
- LibreOffice headless模式
- 自动重试（3次）+ 超时控制（60秒）
- 输出文件名自动处理

### 4. OAuth社交登录

**支持平台**：
- Google OAuth 2.0
- GitHub OAuth

**用户关联逻辑**：
- OAuth ID已存在 → 直接登录
- Email已存在 → 自动绑定OAuth
- 全新用户 → 创建账户

### 5. WebSocket实时进度

**技术特性**：
- 连接管理器（多客户端支持）
- 心跳机制（30秒间隔）
- 自动重连（3秒延迟）
- 降级策略（WebSocket→轮询）
